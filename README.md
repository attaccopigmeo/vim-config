# Vim Configuration

A minimal and practical Vim setup focused on confortable editing and efficient navigation. 

## Features

### UI & Basics
- Line numbers enabled
- Syntax highlighting
- Filetype detection with plugins and indentation
- Highlight current line
- True color support with desert colorscheme

### Indentation
- Tab width: 4 spaces
- Shift width: 4 spaces
- Tabs are converted to spaces

### Search
- Case-insensitive search
- Smart case enabled

### Key Mappings
- Ctrl + n - toggle file tree (NERDTree)
- Ctrl + s - save file
- Ctrl + h - move to left window
- Ctrl + l - move to right window

### Extra
- Mouse support enabled
- System clipboard integration

## Installation
git clone https://github.com/attaccopigmeo/vim-config.git
cd vim-config
cp .vimrc ~/
